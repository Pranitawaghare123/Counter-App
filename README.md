# Counter App with Dark Mode

A responsive Counter Application built using HTML, CSS, and JavaScript. This project allows users to increase, decrease, and reset the counter value with interactive UI feedback and keyboard support.

## Features

* Increase counter value
* Decrease counter value
* Reset counter functionality
* Dark mode and light mode toggle
* Responsive design for mobile and desktop
* Keyboard support using Arrow Keys
* Dynamic warning and success messages
* Smooth UI transitions and hover effects

## Keyboard Controls

* `Arrow Up` → Increase Counter
* `Arrow Down` → Decrease Counter

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Concepts Practiced

* DOM Manipulation
* Event Listeners
* Conditional Statements
* Dynamic Class Switching
* Functions and Reusability
* Keyboard Events
* Responsive Web Design

## Project Goal

The main goal of this project was to practice JavaScript fundamentals by building an interactive and responsive web application with real-time user interactions.
